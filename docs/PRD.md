# Product Requirements (MVP)

## Goal
Recommend the best card before a purchase using predicted category and user priorities.

## Core Flow
1. User opens app/widget before paying.
2. App shows "Use Card X" with confidence and reasons.
3. After transaction posts, app learns from actual category.

## MVP Features
- Location-based merchant detection (POI match)
- Rule-first category prediction
- Weighted reward decision engine
- User preference modes: max points, max cashback, bonus chase
- Recommendation logging for feedback loop

## Out of Scope
- Automatic Apple Pay card switching
- Full bank integrations on day 1
